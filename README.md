# gutentag-world-julia

> Gutentag, World! in Julia

## Run

```
julia -e "using GutentagWorld; println(gutentag_world())"
```


## Install

```
using Pkg; Pkg.add("GutentagWorld")
```


### Usage

## Usage

```sh
julia -e 'using GutentagWorld; println(gutentag_world())'
```

Or in the REPL:

```julia
using GutentagWorld
println(gutentag_world())
```
